# PromptForge

Build powerful AI agents with drag-and-drop simplicity.

## Features

- **Agent Builder** - Visual drag-and-drop builder for agents.md and CLAUDE.md files
- **Preset Library** - Production-ready capabilities and rulesets
- **Live Preview** - See your configuration as you build
- **Export** - Download as .md file ready to use

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- @dnd-kit (drag and drop)
