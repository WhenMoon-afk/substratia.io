# Substratia

Minimal static developer landing page for [substratia.io](https://substratia.io).

Current public purpose:

- developer identity for Substratia
- support contact for `dev@substratia.io`
- current project note for Pocket Universe: Terraformer
- privacy policy and terms for closed testing

## Commands

```bash
npm run check
npm run build
npm run dev
```

The deployable site is generated into `apps/web/dist`.
