# Substratia Site Notes

This repository is intentionally small.

Current purpose:

- static developer landing page for `https://substratia.io`
- support contact for `dev@substratia.io`
- privacy policy and terms for Pocket Universe: Terraformer closed testing

Do not add frameworks, authentication, dashboards, databases, analytics, or old
product-line content unless the project direction explicitly changes.

Useful commands:

```bash
npm run check
npm run build
npm run dev
```

The static output is `apps/web/dist`.
